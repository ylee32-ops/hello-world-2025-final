# Animal Kingdom Game

A political simulation game where you lead the animal kingdom through 10 critical decisions. Balance four metrics (Treasury, Public Trust, International Reputation, Social Stability) while managing 12 different animal communities.

## Quick Start

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`

## How to Play

**Drag LEFT or RIGHT** to choose between two options, or **click the orange buttons**. Keep all 4 metrics above zero to survive all 10 events.
